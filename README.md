### 🚀 About Me

Building is my favourite way of learning. I enjoy solving real-world challenges by combining technology with business. Curious by nature and persistent by choice, I'm always exploring new ideas and building AI-powered tools, automation workflows, and full-stack applications that make everyday tasks smarter and simpler. I believe persistence beats perfection, and innovation isn't just about having ideas-it's about bringing them to life!

🔭 &nbsp;I'm currently working on **Enhancing a real-time chat platform with new collaboration features and improving a stock market application with live data, analytics, and learning tools.**  
🌱 &nbsp;I'm currently learning **Scalable System Design, Cloud Architecture, Production-Ready AI Applications, Advanced Automation Workflows, and Modern Deployment Practices**  
👯 &nbsp;I'm looking to collaborate on **Full-Stack applications, AI &amp; Automation tools, Developer tools.**  
🤔 &nbsp;I'm looking for help with **Cloud architecture, scalable system design, and production-ready deployments.**  
💬 &nbsp;Ask me about **React, JavaScript, REST APIs, Python, DSA, Automation &amp; Problem Solving**  
😄 &nbsp;Pronouns: **She/Her**
